## 2. 核心模块说明

### 2.1 Base Scraper (skills/base/scraper.py)
- **职责**: 提供抓取器基类，封装通用功能
- **关键方法**:
  - `get_headers()`: 随机 User-Agent
  - `random_delay()`: 随机延迟防封
  - `validate_data()`: 数据验证

### 2.2 Dianping Scraper (skills/dianping/scraper.py)
- **挑战**: 大众点评无API，需要反爬
- **方案**:
  - Playwright 模拟浏览器
  - 随机 viewport 和 user-agent
  - 请求频率控制 (3-8秒随机延迟)
  - Cookie 持久化
- **抓取字段**:
  - 评分、评论数、口味/环境/服务分
  - 最新评论内容
  - 门店排名

### 2.3 Delivery Scraper (skills/delivery/scraper.py)
- **支持平台**: 美团、饿了么、京东外卖
- **抓取字段**:
  - 有效订单量
  - 转化率、客单价
  - 商圈排名
  - 门店评分

### 2.4 Feishu Notifier (skills/feishu/notifier.py)
- **功能**: 飞书机器人消息推送
- **特性**:
  - 签名验证
  - 富文本卡片
  - @指定用户
  - 多平台数据汇总展示

### 2.5 Scheduler (skills/scheduler/cron.py)
- **调度策略**:
  - 评价平台: 每2小时 (10:00-20:00)
  - 外卖平台: 每30分钟 (午/晚餐时段)
- **实现**: 基于 `schedule` 库

## 3. 反爬策略

1. **请求头伪装**: 随机 User-Agent
2. **请求频率控制**: 3-8秒随机延迟
3. **浏览器模拟**: Playwright + 随机 viewport
4. **失败重试**: 指数退避重试机制
5. **IP轮换**: 支持代理池（预留接口）

## 4. 数据验证

- **准确率目标**: ≥ 98%
- **验证机制**:
  - 必填字段检查
  - 数据类型验证
  - 异常值检测
  - 历史数据对比

## 6. 扩展性设计

- **模块化**: 每个平台独立 Scraper，易于新增
- **配置化**: YAML 配置，无需改代码
- **插件化**: 支持自定义数据处理插件

## 7. 监控与告警

- **日志**: 结构化日志，支持日志轮转
- **告警**: 抓取失败自动飞书告警
- **指标**: 成功率、延迟、数据量

## 8. 安全考虑

- 配置文件敏感信息隔离
- Webhook URL 和 Secret 不硬编码
- 请求签名验证
- 错误信息脱敏
